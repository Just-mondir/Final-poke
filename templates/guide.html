Follow these steps to set up your Google Sheets automation

1Create a Google Cloud Project

First, you need to create a Google Cloud project and enable the necessary APIs:

  • Go to the Google Cloud Console (console.cloud.google.com)
  • Create a new project or select an existing one
  • Enable the Google Sheets API and Google Drive API

2Create a Service Account

Create credentials that will allow the automation to access your Google Sheets:

  • In the Google Cloud Console, go to IAM & Admin ‚Üí Service Accounts
  • Click "Create Service Account"
  • Give it a name like "sheets-automation"
  • Skip the optional steps and click "Done"

3Generate JSON Credentials

Download the credentials file that the automation will use:

  • Click on your newly created service account
  • Go to the "Keys" tab
  • Click "Add Key" ‚Üí "Create new key"
  • Select "JSON" format
  • The JSON file will be automatically downloaded
‚ö†Ô∏è Important: Keep this JSON file secure! It contains credentials that can access your Google services. Never share it publicly or commit it to version control.

4Share Your Google Sheet

Give the service account access to your Google Sheet:

  • Open your JSON credentials file and find the "client_email" field
  • Copy the email address (it looks like: your-service@project-name.iam.gserviceaccount.com)
  • Open your Google Sheet
  • Click "Share" in the top-right corner
  • Paste the service account email
  • Set permissions to "Editor"
  • Click "Send" (you may see a warning that it can't send email - that's fine)

5Prepare Your Sheet Structure

Make sure your Google Sheet has the correct structure:

  • Column F (6): URLs to scrape
  • Column G (7): Grader name (e.g., "PSA", "BGS")
  • Column H (8): Grade (e.g., "10", "9.5")
  • Columns L-O (12-15): Will be filled with individual prices
  • Column P (16): Will be filled with the average price
  • Starting row: Data should start from row 9

6Upload and Run

Now you're ready to use the automation:

  • Go back to the main page
  • Upload your JSON credentials file
  • Enter your Google Sheet name (exact name as shown in Google Sheets)
  • Click "Start Automation"
  • Monitor the progress in real-time

7Troubleshooting

If you encounter issues, check these common problems:

  • Permission denied: Make sure you shared the sheet with the service account email
  • Sheet not found: Double-check the sheet name spelling
  • Invalid JSON: Ensure you downloaded the correct credentials file
  • No data found: Verify your sheet has data in the correct columns
